综述
===================================  
基于Asp.net，SqlSever2012,VS2010编写大学生竞赛系统。</br>
竞赛系统面向学生用户，教师用户，教委用户，学校用户分别实现不同的功能。</br>
熟练使用了C#面相对像编程和PowerDesigner设计UML类图解决同步交互的问题和使用C#的LinqtoSql类对数据库访问和操作高效封装。</br>
